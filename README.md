# Petty Cash System

A simple and efficient system for managing petty cash transactions and expenses.

## Features

- Track petty cash transactions
- Manage expense categories
- Generate expense reports
- User-friendly interface

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
git clone https://github.com/jackdevs-lab/petty-cash-system.git
cd petty-cash-system
npm install
```

### Usage

```bash
npm start
```
